React Memory Game

Jujyfruit's Party is a game developed with React.js. The player is asked to click on each character's card. After the user clicks on a card, however, all cards will get shuffled, and the player needs to make sure that they do not click the same card twice. Inspired by Gregg Araki's film 'Nowhere' (1997).
