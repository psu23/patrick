Youjuke (Version 1.0)

info

Youjuke (1.0)

Youjuke is a music app that allows groups of visitors to compile a shared playlist and listen to it together, live. Anyone can add a track to the playlist, but it is up to all users as a whole to decide the order of the playlist through voting. Like radio, the playlist is updated live for everybody accessing it. A concept for both virtual and in-person parties.

Technologies Used
